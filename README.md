## 🔒 Project Overview

**IRONCORE** is a premium, high-conversion landing page built for elite fitness coaching and biological optimization protocols. The interface follows a "Digital Vault" aesthetic—utilizing glassmorphism, 3D spatial depth, and interactive HUD elements to create a sophisticated, high-end user experience.

This project is engineered with a focus on:
- **Premium Aesthetics**: Dark mode by default with tonal layering and neon-primary accents.
- **Vibrant Motion**: A combination of GSAP-powered parallax effects and Framer Motion orchestrations.
- **Frictionless UX**: Custom HUD-style cursor feedback and seamless section transitions.

---

## 🛠 Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Inter (Sans), Poppins (Display), JetBrains Mono (Technical)
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## ✨ Features

### 1. The Digital Vault UI
- **Spatial Grid System**: A persistent 3D grid background that responds to user movement.
- **HUD Cursor Engine**: A custom interactive pointer that provides real-time feedback when hovering over interactive "Nodes".
- **Glassmorphic Components**: High-fidelity surfaces with backdrop-filters and grain overlays for a premium tactile feel.

### 2. Operational Protocols (Programs)
- Interactive 3D cards with hover-sensitive lighting and Z-axis depth.
- Clear technical metadata for Difficulty, Intensity, and Duration benchmarks.

### 3. Infinite Force Hero
- High-impact typography with editorial-style spatial layering.
- Floating 3D assets and scanning HUD animations to capture immediate attention.

### 4. Precision Contact System
- "Transmission" based application form with state management and glass-flare aesthetics.
- Capacity-tracking urgency badges for conversion optimization.

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm / pnpm / yarn

### Installation
1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the portal at `http://localhost:3000`.
